CoreML Benchmark — M2 MacBook Air (Apple Silicon ANE)

MobileNetV2 Inference Latency — Compute Unit Comparison

2.55ms CPU Only — avg
1.96ms CPU + GPU — avg
0.63ms ANE (ALL) — avg
2.64ms CPU Only — P99
3.83ms CPU+GPU — P99 ⚠️
0.72ms ANE — P99 ✓
Key finding: CPU+GPU average (1.96ms) appears faster than CPU-only (2.55ms) — but P99 spikes to 3.83ms due to GPU scheduling contention with the render pipeline. ANE delivers 4× speedup over CPU-only with near-zero P99 variance (0.09ms spread). At Vision Pro's 120Hz frame budget (8.3ms), ANE inference consumes 0.63ms — leaving 7.70ms headroom for concurrent real-time workloads. Average latency hides the real story. P99 is what matters for real-time systems.
